# Детский массажист Оксана — лендинг

Небольшой приветственный лендинг с формой записи.

## Как запустить

Это статический сайт. Можно открыть файл `index.html` двойным кликом.

Если нужен локальный сервер (чтобы всё работало одинаково в разных браузерах), варианты:

- **VS Code**: расширение *Live Server* → `Go Live`
- **Python (если установлен)**:

```bash
python -m http.server 5173
```

Откройте в браузере `http://localhost:5173`.

## Что внутри

- `index.html` — страница лендинга
- `styles.css` — стили
- `script.js` — функционал формы (валидация + сохранение заявок в `localStorage`)
- `Oxana.jpg` — фото для hero-блока
- `Код.md` — исходник, который был взят за основу

## Примечание про заявки

Сейчас отправка сделана как **демо**: заявки сохраняются в браузере (кнопка «Показать сохранённые заявки» внизу).
Если нужно — подключу реальную отправку в Telegram/почту/CRM.
